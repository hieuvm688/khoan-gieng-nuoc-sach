# Khoan Giếng Nước Sạch

Website cung cấp thông tin và báo giá dịch vụ khoan giếng nước ngầm chuyên nghiệp tại Miền Trung.

Giao diện trang đích tĩnh (Landing Page) sử dụng HTML, CSS và JavaScript nguyên bản với hiệu ứng hiện đại.

## Khu vực phục vụ
Thanh Hoá, Ninh Bình, Hà Nam, Nghệ An, Hà Nội và các tỉnh lân cận.

## Đặc điểm nổi bật
- Giới thiệu dịch vụ rõ ràng, chuyên nghiệp.
- Giao diện thân thiện và tối ưu trên mọi màn hình (Responsive).
